## üìã Description

This is a functional clone of the famous mobile game Flappy Bird, created using Unity 2022.3.9f1 and programmed in C#. The game implements all essential mechanics: bird control, flight physics, procedural obstacle generation, scoring system, and collisions.

## üéÆ How to Play

### Controls
- **Left Mouse Click** or **Space**: Make the bird fly
- **R**: Restart the game after dying

### Objective
- Fly between obstacles (pipes) without colliding
- Each obstacle passed adds 1 point to your score
- The game ends when you collide with an obstacle or the ground
